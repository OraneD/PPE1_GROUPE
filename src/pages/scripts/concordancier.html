Scripts concordanciers

Pour obtenir une vue sur les cooccurrences du mot analysé, nous avons créé un concordancier pour chaque URL dans chaque tableau. Le concordancier affiche le mot cible et son entourage c'est-à-dire le contexte gauche et le contexte droit.

Les concordanciers ont été générés avec le script Bash. Ce script est le complément du script tableau et il est donc appelé lors de l'exécution du script tableau. Son résultat est affiché dans la colonne concordances de nos tableaux.

Le script concordance considère le texte comme un ensemble d'unités : le mot cible est une unité séparé par des parenthèses, le contexte gauche est une unité et le contexte droit est une unité. Le mot cible provient de la variable mot du script tableau. Étant donné qu'il s'agit d'une expression régulière (avec des parenthèses), nous avons dû compter le nombre des parenthèses qu'a pris le mot cible afin de retrouver le contexte droit. C'est pour cela que nous avons deux scripts différents : les mots cibles des corpus anglais, français et du corpus serbe "štreber" ("nerd") ont le même nombre de parenthèses dans l'expression régulière et un même script concordance a pu être appelé pour générer les tableaux. Or, l'expression régulière pour le mot "gik" ("geek") en serbe contient une parenthèse en plus donc un deuxième script a été utilisé.


Sur cette page, vous trouverez les deux scripts :

Script pour les corpus français, anglais et le corpus "štreber" ("nerd") serbe
Cliquez pour télécharger le script ci-dessous

#!/usr/bin/bash



fichier_text=$1

motif=$2


if [[ $# -ne 2 ]]

then

               echo "Ce programme demande exactement deux arguments."

               exit

fi


if [[ ! -f $fichier_text ]]

then

         echo "le fichier $fichier_text n'existe pas"

         exit

fi


if [[ -z $motif ]]

then

         echo "le motif est vide"

         exit

fi


echo "

<!DOCTYPE html>

<html lang="en">

<head>

    <meta charset="UTF-8">

    <title>Concordance</title>

</head>

<body>

<table>

<thead>

    <tr>

        <th class="has-text-left">Contexte gauche</th>

        <<th>Cible</th>

        <<th class="has-text-right">Contexte droit</th>

    <</tr>

</thead>

<tbody>

"


grep -E -o "(\w+\W+){0,5}($motif)(\W+\w+){0,5}" $fichier_text | sed -E "s/(.*)($motif)(.*)/<tr><td>\1<\/td><td>\2<\/td><td>\4<\/td><\/tr>/"


echo "

</tbody>

</table>

</body>

</html>

"
